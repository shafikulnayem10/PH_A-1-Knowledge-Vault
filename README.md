
# 📘 Programming Hero – Web Development Course  
## Assignment 1 (PH_A-1) – Solution

This repository contains my solution to **Assignment 1** of the Programming Hero Web Development Course.

The project was built using **pure HTML5 and CSS3**, focusing on mastering the fundamentals of web development without relying on any external libraries or frameworks.

---

## 🔗 Live Demo

👉 https://knowledge-vault-kv.netlify.app/

## 🎯 Project Goal

The main objective of this assignment was to build a solid foundation in frontend development by:

- Understanding how the web structure works using HTML  
- Applying clean and maintainable styling using CSS  
- Learning layout, spacing, and alignment techniques  
- Writing structured and semantic code for better readability  

This project emphasizes **clarity, structure, and proper use of core web technologies**.

---

## ✨ Key Features

- 📄 Well-structured semantic HTML layout  
- 🎨 Clean and organized CSS styling  
- 📐 Proper spacing, alignment, and visual hierarchy  
- 📱 Basic responsive design considerations  
- 🧩 Readable and beginner-friendly code structure

## 🧠 How It Works (Technical Breakdown)

- Used **semantic HTML tags** (`header`, `section`, `footer`) to improve structure and accessibility  
- Applied **CSS styling** for layout control, spacing, and typography  
- Organized styles in a clean and maintainable way  
- Focused on **visual consistency and readability** across the page
